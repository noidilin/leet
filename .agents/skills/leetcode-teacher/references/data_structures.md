# Deprecated Data Structures Index

This file is kept only as a legacy compatibility pointer for older notes.

Do not use it as the primary reference anymore because it mixes foundations with later problem-solving frameworks.

Use these staged references instead:

- `data_structure_foundations.md` -> arrays, linked lists, circular array, queue, stack, hash tables, trees, heaps, and graph basics.
- `linear_problem_frameworks.md` -> two pointers, sliding window, binary search, prefix sum, and difference array.
- `stack_queue_frameworks.md` -> classic stack or queue exercises, monotonic stack, and monotonic queue.
- `tree_recursion_frameworks.md` -> recursion through binary trees, level-order thinking, BST, and trie.
- `graph_algorithms.md` -> graph traversal review, cycle detection, topological sort, union-find, MST, Kruskal, and shortest path frameworks.

If an older note mixes a structure with a framework, prefer the staged file that matches the learner's current course stage.
