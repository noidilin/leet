# Binary Tree

Binary tree is a must-focus topic, especially traversal—the key to mastering recursive thinking. All recursive algorithms are essentially binary tree traversal: recursive traversal leads to backtracking/DP; level-order traversal leads to BFS/Dijkstra shortest path algorithms.
