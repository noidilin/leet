# Hash Table

- Hash tables are a very classic and commonly used data structure. It is essential to master their principles and understand the reasons for hash collisions and the approaches to solving them using chaining and linear probing.
- Hash tables can be combined with arrays and doubly-linked lists to form more powerful structures like LinkedHashMap and ArrayHashMap, adding new capabilities. Know their principles so you can apply them in the right scenarios.
